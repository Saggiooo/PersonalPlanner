# Organizer

App desktop locale per macOS costruita con `Tauri + React + TypeScript + SQLite`.

## Cosa include

- board kanban per progetto
- vista timeline per leggere i task nel tempo
- panorama misto di tutti i progetti
- persistenza locale in `sqlite:organizer.db`

## Avvio

```bash
npm install
npm run tauri dev
```

## Build frontend

```bash
npm run build
```
